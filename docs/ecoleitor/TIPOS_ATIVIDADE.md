# EcoLeitor – Tipos de Atividade

## ditado_oral
Escrita produzida a partir de uma palavra falada pelo professor.

## associacao_imagem
Escrita produzida a partir da identificação de uma imagem. Avalia reconhecimento da imagem, nomeação mental e hipótese de escrita.

## banco_palavras
Escrita vinculada ao banco temático de palavras construído com a turma.

## texto_livre
Produção escrita espontânea ou autoral.

## Regra metodológica
Os registros avulsos devem ser separados dos ditados orais e do banco de palavras para não misturar diferentes condições de produção da escrita.
